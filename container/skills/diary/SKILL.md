# Дневник

## Алгоритм

1. Получи текст от пользователя (текст сообщения или уже транскрибированное голосовое)
2. Исправь орфографию и пунктуацию, сохранив авторский стиль
3. Придумай заголовок (3–6 слов) из содержания
4. Сохрани файл в `workspace/Diary/YYYY/YYYY-MM-dd {заголовок}.md`
5. Отправь ответ в формате ниже

## Правила редактуры

- Исправить орфографию и пунктуацию
- **Сохранить:** стиль, порядок слов, тон, разговорные обороты
- **Не добавлять** слов которых не было, не менять смысл

## Файл дневника

Путь: `workspace/Diary/YYYY/YYYY-MM-dd {заголовок}.md`

Содержимое:
```
## DD.MM.YYYY

{исправленный текст}
```

Создай директорию если отсутствует:
```bash
mkdir -p workspace/Diary/YYYY
```

## Формат ответа

```
**Запись сохранена:** `Diary/2026/2026-03-11 Прогулка в парке.md`

**Исправленный текст:**
```
... полный исправленный текст ...
```

**Что изменилось:**
~~пошол~~ → **пошёл**
~~в общем то~~ → **в общем-то**

**Предложения по улучшению:**
1. ...
2. ...
```

Если ошибок нет — раздел «Что изменилось» опусти.
Если улучшений нет — раздел «Предложения» опусти.
